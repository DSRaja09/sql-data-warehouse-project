# 📊 Data Warehouse Project (SQL Server)
### 🚀 Overview

This project is a Data Warehouse implementation built using Microsoft SQL Server, with development and management done through VS Code.

The goal of this project is to design and implement a structured data warehouse that can efficiently store, organize, and analyze large volumes of data for reporting and decision-making.

### 🛠️ Tech Stack

Database: Microsoft SQL Server

Editor: Visual Studio Code (VS Code), draw.io

Language: SQL

### 📦 Features

✅ Data warehouse schema design (Star/Snowflake)

✅ ETL (Extract, Transform, Load) processes

✅ Data cleaning and transformation

✅ Aggregations and analytical queries

✅ Optimized queries using indexing

✅ Views and stored procedures for reporting
